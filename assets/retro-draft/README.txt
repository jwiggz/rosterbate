NBA retro draft-room placeholders

Drop your assets into this folder using these filenames:

- nba-retro-room-placeholder.jpg.png
  Full retro room background image for the NBA draft room.
  The main TV is already inside this artwork.

- nba-tv-loop-01.gif
- nba-tv-loop-02.gif
- nba-tv-loop-03.gif
- nba-tv-loop-04.mp4
  Favorite game clips that will be randomly selected on load.

Notes:

- MP4 or WebM will perform better than GIF when possible.
- You do not need to use every slot.
- The NBA draft room will try the listed files in random order and fall back to the built-in placeholder if none load.
